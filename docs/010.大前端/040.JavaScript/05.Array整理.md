## 对象方法

| 属性        | 描述                             |
| ----------- | -------------------------------- |
| constructor | 返回对创建此数组函数的引用       |
| length      | **设置**或返回此数组中元素的数目 |
| prototype   | 获得向对象添加属性的方法         |
