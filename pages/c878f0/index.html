# 清除 opacity 子元素继承问题

问题描述

父元素在使用 opacity 属性时，子元素会继承该属性

解决方法

解决方案并不唯一，就比如脱离文档流，或者把子级元素变为与 opacity 属性元素同级，此处仅列举最简单的处理方法

使用background: rgba(255,0,0,0.5) 代替opacity ，切不会有属性继承问题
